1. Explain the difference between inline, embedded, and external CSS.

Inline CSS

  • It written inside the style attribute of an element and changes only that one element.

    • Embedded CSS

  • It placed inside a style tag in the HTML file and styles the whole page

    • External CSS

  • It is in a separate file and is linked to the HTML, making it easy to style many pages at once.

    • External CSS is best for big projects, while inline and embedded CSS are good for small changes.
