1. List out the different ways to select an item.

By Element (p)
By ID (id)
By Class (class)
By Nesting (div)

2. What is the difference between an ID and a Class?

ID: Only use for one unique thing
Class: Can use multiple times

3. What is the difference between a SPAN tag and a DIV tag?

SPAN: Use for layout and big text.

DIV: Use for small text.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

DIV: Mainly used for layout and styling (No meaning)
Semantic tags: Define specific parts of a webpage
HEADER: Top section of semantic tags
MAIN: Main content semantic tags
FOOTER: Bottom section semantic tags

5. What are the different ways to select a color?

Hex codes (#ff0000)
RBG (rgb(255, 87, 51))
RGBA (adds transparency)
HSL (hue, saturation, lightness)
HSLA (adds transparency)
