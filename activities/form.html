Questions:

1. What are different ways to style your form?

You can style your form using basic CSS, frameworks like Bootstrap or Tailwind, Flexbox for layout, animations for interactivity, and custom fonts and colors for a unique look.

2. Why does the professor like to use flex?

Because it makes layout design easier and more flexible. Flexbox allows elements to automatically adjust their size and position based on the available space, making it great for responsive design. It simplifies alignment, centering, and spacing without requiring extra CSS tricks, making it an efficient way to structure forms and other UI elements.
