When do you use a table?

Use a table to organize and compare data clearly, especially when dealing with structured information or numbers.

When do you not use a table?

Avoid using a table when presenting simple text, explaining concepts, or when a list, paragraph, or visual would be clearer and easier to read.

When do you need to do extra work for table accessibility?

Extra work is needed for table accessibility when it has complex headers, merged cells, or non-text elements, requiring tags and labels for screen readers.
