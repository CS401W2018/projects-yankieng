Why do we need a viewport meta tag?

The viewport meta tag ensures your webpage adjusts to different screen sizes, making it look good on all devices.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The last declared rule takes priority unless a media query or a more specific selector overrides it.

How do you change the CSS for a web page displayed on a phone?

Use a media query to change CSS when the screen is small, like on a phone.
