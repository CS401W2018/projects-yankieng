Why is the viewport meta tag important?

The viewport meta tag makes web pages look good on all screen sizes. It stops pages from being too small or zoomed out, making them easy to read and use on mobile devices.

How can you adjust the CSS for printing?

Use @media print in CSS to adjust styles for printing. Hide unwanted parts like the 'nav' and 'footer', remove backgrounds, adjust text size for readability, and prevent content from breaking across pages. This makes prints cleaner and easier to read.

When would you want to adjust the CSS for printing?

I would adjust CSS for printing when you want a web page to look clean and easy to read on paper. This is useful for printing articles, receipts, reports, or invoices by removing menus, ads, or background colors to save ink and improve readability.
