What you learned about Flex and CSS.

I learned that flex in CSS makes layouts flexible and easy to align. It uses a flex container to control child elements with properties like `justify-content`, `align-items`, and `flex-flow`.

Why do you need to have an accessible website? What does that mean?

Because an accessible website allows everyone, including people with disabilities, to use it easily. It improves user experience, reaches more people, and follows legal guidelines.
