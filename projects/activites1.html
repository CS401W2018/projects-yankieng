1. Internet

The Internet is a global network that connects computers and devices which allows people to share information and access services like websites and email. It works through standard communication rules and links networks worldwide.

2. Web

The Web is a part of the Internet that allows people to access and share information using websites. It works through browsers and links content using technologies like HTTP. The Web lets users view and interact with text, images, videos, and more online.

3. How are the Internet and Web different?

The difference between Internet and Web is Internet is the foundation and the Web is one of the services built on top of it.

4. ASCII and Unicode

ASCII is a character encoding system that uses 7 bits to represent 128 characters, including English letters, numbers, and basic symbols. It's simple but limited to English. Therefore, someone invented the Unicode which is a more advanced system that supports characters from all languages and symbols, using variable encoding like UTF-8. Itâ€™s much more versatile and includes everything in ASCII, plus many more characters.

5. What is UTF-8 ?

UTF-8 is a system for encoding text that supports all languages and symbols by using 1 to 4 bytes per character. It's efficient, works with ASCII, and is widely used on the Internet.

6. URL

A URL is like an Internet address for finding specific content and to access resources on the Internet.

7. Domain Name

A domain name is a user-friendly website address that simplifies access by replacing complex numerical IP addresses.

8. HTTP

HTTP is a system that helps your web browser talk to websites so you can see pages and get information.

9. HTTPS

HTTPS is a secure version of HTTP. It keeps your information safe by encrypting data between your browser and the website.

10. IP Address

An IP address is a unique number that identifies a device on the Internet. It helps devices find and communicate with each other. There are two types of IP address, which are IPv4 and IPv6.

11. DNS

DNS is a system that turns website names into numbers (IP address) that computers use to find websites. It makes browsing easier by letting you use names instead of numbers.

12. W3C Recommendations

W3C Recommendations are rules set by the World Wide Web Consortium. These rules help websites work properly on different computers and phones, making the web easier to use for everyone.

13. XML, XHTML, HTML, HTML5

HTML is the language used to make web pages. XHTML is a more strict version of HTML. HTML5 is the newest version with extra features like video support. XML is used to store and share data, not for showing content on websites. Each one of them has a special role in building or organizing things on the web.
